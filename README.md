# ContactWebApp — Менеджер контактов с пагинацией

Простое одностраничное приложение для управления списком контактов: добавление, редактирование, удаление и пагинация.  
Разработан как учебный проект для отработки навыков веб-разработки на **.NET 8**, **PostgreSQL**, **Entity Framework Core** и **React с Bootstrap 5**.

## Возможности

- Просмотр списка контактов с пагинацией
- Добавление новых контактов (имя, email)
- Редактирование существующих контактов
- Удаление контактов
- Адаптивный интерфейс с Bootstrap 5
- Single Page Application (SPA)
- Полная интеграция с REST API на .NET 8

---

## Технологии

### Backend
- **ASP.NET Core Web API (.NET 8)**
- **Entity Framework Core** + **Npgsql** (PostgreSQL)
- **JSON API** для взаимодействия с фронтендом
- **Swagger** для тестирования API

### Frontend
- **React** (с использованием Hooks и функциональных компонентов)
- **Bootstrap 5** для стилей и компонентов UI
- **Axios** для взаимодействия с бэкендом
- Модульная структура компонентов

---

## Требования

- [.NET 8 SDK](https://dotnet.microsoft.com/download/dotnet/8.0)
- [PostgreSQL 15+](https://www.postgresql.org/download/)
- Браузер с поддержкой ES6+ (Chrome, Firefox, Edge)
